# Технический паспорт изделия
## Изделие: Мнемоключ МК-7

**Назначение:** аппаратный маркер сеанса нормализации и подписи набора документов для «Сонаты-2041».  
**Статус:** учебный контур, промышленная сертификация не проводилась.

### 1. Идентификация
- Полное имя: **Мнемоключ МК-7**  
- Серийный номер: **7-041-118** (семь-ноль-сорок один-сто восемнадцать)  
- Год сборки: **2041** (две тысячи сорок первый)  
- Владелец: АО «СевКварц»

### 2. Принцип работы (кратко)
МК-7 создаёт “отпечаток набора”:
- читает список документов (по внутренним идентификаторам),
- формирует подпись набора,
- передаёт подпись модулю «Сольвейг» для фиксации в индексе.

В документации подпись иногда называется “мнемосигнатура”.

### 3. Совместимость
- Сервер: **«Лира-2»** (корпус Б, подвал, стойка R5)  
- Модуль нормализации: **«Сольвейг»**  
- Алгоритм семантики: **«Резонанс-3»**

### 4. Режимы
- Режим А: “подпись набора” (основной)  
- Режим Б: “проверка целостности” (контрольный)  
- Режим В: “сверка регламента 90 дней” (служебный, только для инженера)

### 5. Ограничения и правила хранения
- Не выносить за пределы корпуса Б.  
- При перерыве более **48** (сорока восьми) часов сдавать в шкаф дежурного архивиста.  
- Рабочие индексы живут **90** (девяносто) дней с момента обновления; МК-7 лишь подтверждает дату обновления, но не хранит данные.

### 6. Примечание для тестов поиска
Для проверки точного и семантического поиска рекомендуется включать в тексты обе формы:
- “МК-7”,  
- “мнемоключ”,  
- “аппаратный маркер”.

Это не разные сущности: это один и тот же объект.
